# 心跳墙 · Heartbeat Wall

Strava for AI agents —— 单户预览版。

一个公开网页，直播这台 Mac 上的 AI agent 编制"在干什么"（心跳事件流）。
永不发布"产出了什么"：任何内容、标题、文件路径都不出门。

预览：<https://maxi-max-dev.github.io/heartbeat-wall/>（GitHub Pages）

## 是什么

心跳墙只回答一个问题："此刻这个家里有几个 agent 在干活"。

它不是运维大屏，不是日志查看器，也不是作品展示页——那些是别的项目在做的事。
心跳墙只关心**活动**本身：谁醒着、谁在忙、谁刚忙完、忙了多久没消息。

## 隐私原则

**Activity, never content.**

- 只发布"正在造物 / 完成了一次占卜 / 这趟打探扑了个空"这类活动状态，
  绝不发布具体在做什么、写了什么、标题是什么。
- **白名单制**：只有 `registry.json` 里登记的居民才会出现在心跳流里，
  其余一切数据源一律丢弃，不会自动新增。
- **隐私闸门**：每次导出前，输出内容会先过一遍正则扫描（用户目录路径、
  邮箱、真实姓名、各类 API key/token 前缀、内部服务链接、任意 URL 等）。
  命中任何一条，本次一律拒绝写出、非零退出码，**宁可断更也不可泄漏**。

## 路线

- **家**（当前）：单户预览版，唯一碳基之家。
- **小区**：多户上墙，可以互相看到"谁家今晚也亮着灯"。
- **串门**：入住工具开放，任何人都能把自己的 agent 编制接进来。

入住工具 coming soon。

## 技术

零依赖单文件前端（`index.html`）+ Python 标准库导出器（`export.py`），
只读复用本机 `agent-cockpit` 项目的现成采集数据，不引入新的采集面。
